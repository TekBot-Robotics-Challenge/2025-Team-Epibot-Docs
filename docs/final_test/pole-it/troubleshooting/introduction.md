# Troubleshooting index

We collected common problems, fixes and notes while assembling, calibrating and running the Dofbot. This index links all troubleshooting pages we keep in this repo.

Pages

- `Assembly.md` — assembly steps, common hardware issues and mobile app calibration
- `Documentation_reading.md` — lessons learned when using vendor docs and tutorials
- `Initial_setup.md` — first steps and initial calibration checklist
- `Setup.md` — practical setup issues and fixes (I2C, camera, dependencies)
- `Tutorials.md` — tutorials we used and how we applied them
- `VM_installation.md` — importing the VM and installing VSCode
- `Sorting_positions.md` — how we captured servo positions and derived joint poses for the bins
- `Camera_calibration.md` — checkerboard calibration notebook workflow and issues
